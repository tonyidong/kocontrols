Requirements:
	.Net 4.0
The KO.Controls library currently consists of one control: AutoSuggest textbox.
You may see how it works by starting the ControlTestApp.exe.
To try it delete the displayed text in the textbox and start typing to select another city. 
To use the control you really need the source code. Please go ahead and download it and take a look at the example provided.
